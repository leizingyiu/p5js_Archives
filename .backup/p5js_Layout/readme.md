# Layout.js _  在 p5.js 中，快速排列文本

预览效果请浏览demo https://openprocessing.org/sketch/1597189 

## 范例

```javascript
some_text=new Layout('some text', {
        font: titleFont,
        margin: 12, padding: 2, spacing: 4,
        left: 0, top: 0, w: width, h: height, fillCell: 'none'
    });
some_text

```
